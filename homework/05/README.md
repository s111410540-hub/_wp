# 墨客部落格 - 版本演進記錄

一個從基礎到現代化的部落格系統演化過程。

---

## 📊 版本總覽

| 版本 | 代號 | 端口 | 主要特徵 |
|------|------|------|----------|
| **ver1.0** | 基礎版 | 3000 | 純功能實現，基本 CRUD |
| **ver2.0** | 現代化 | 3001 | SPA 路由，用戶系統，精選推薦 |
| **ver3.0** | 旗艦版 | 3002 | 會員系統，下拉刷新，無限滾動，極致閱讀體驗 |

---

## 🔄 ver1.0 → ver2.0 演進

### 資料庫結構

| ver1.0 | ver2.0 |
|--------|--------|
| 僅 `posts` 資料表 | 新增 `users` 用戶表 |
| 單純標題/內容/時間 | 新增作者、標籤、封面圖、閱讀時間 |
| 無關聯性 | 新增 `likes` 按讚表，支援用戶互動 |

### 功能對比

| 功能 | ver1.0 | ver2.0 |
|------|--------|--------|
| 頁面路由 | 單頁（列表/詳情切換） | SPA 客戶端路由 |
| 設計風格 | 基礎內聯 CSS | Medium 風格專業 UI |
| 用戶系統 | 無 | 作者資料頁面、個人主頁 |
| 文章精選 | 無 | 精選文章模塊 |
| 推薦系統 | 無 | 按讚數排序推薦 |
| 響應式設計 | 無 | 完整適配移動端 |
| 外部 API | 無 | 頭像生成 API |

### 程式碼改進

```
ver1.0:  ~89 行 server.js + 內嵌 HTML/CSS/JS (~109行)
ver2.0: ~256 行 server.js + 分離的 HTML/CSS/JS 檔案
```

### 新增 API 端點

| 端點 | 說明 |
|------|------|
| `GET /api/posts/featured` | 取得精選文章 |
| `GET /api/posts/recommended` | 取得推薦文章 |
| `GET /api/posts/user/:username` | 取得用戶個人文章 |
| `POST /api/likes/:postId` | 切換按讚狀態 |
| `GET /api/users/:username` | 取得用戶資料 |
| `PUT /api/posts/:id` | 更新文章 |

## 🔄 ver2.0 → ver3.0 演進

### 核心功能新增

| 功能 | 說明 |
|------|------|
| **會員系統** | 完整的登入/註冊功能 |
| **下拉刷新** | 頁面頂部下拉即可刷新文章列表 |
| **無限滾動** | 滚动到底部自動加載更多文章 |
| **返回頂部按鈕** | 滚动超過 300px 時顯示 |
| **浮動操作欄** | 文章閱讀時底部浮動點讚/分享按鈕 |
| **完整文章顯示** | 支援 Markdown 語法解析（標題、列表、引用） |

### 會員系統功能

| 功能 | 說明 |
|------|------|
| **註冊頁面** `/register` | 用戶名（3-20字）、顯示名稱、密碼（至少6字） |
| **登入頁面** `/login` | 用戶名/密碼輸入 |
| **用戶選單** | 頭像下拉：我的主頁、寫文章、登出 |
| **權限控制** | 寫文章需先登入，發布自動綁定作者 |
| **測試帳戶** | 用戶名 `writer` 密碼 `writer123` |

### 使用者體驗提升

| 改進點 | ver2.0 | ver3.0 |
|--------|--------|--------|
| 文章閱讀 | 基礎排版 | 增大行距(2x)，提升閱讀舒適度 |
| 觸控支援 | 基礎支援 | 完整下拉刷新手勢 |
| 導航體驗 | 固定返回 | 智能浮動按鈕 |
| 載入狀態 | 簡單文字 | 動畫載入指示器 |
| 文章內容 | 純文字 | Markdown 語法支援 |

### 前端互動改進

```javascript
// ver3.0 新增功能

// 1. 下拉刷新偵測
document.addEventListener('touchstart/touchmove/touchend', ...)

// 2. 無限滾動偵測
window.addEventListener('scroll', () => {
  if (isNearBottom) loadMorePosts();
})

// 3. 文章浮動按鈕
window.addEventListener('scroll', () => {
  if (headerOutOfView) showFloatingActions();
})

// 4. 會員系統
function saveUser(user) {
  localStorage.setItem('currentUser', JSON.stringify(user));
}
```

### 新增 API 端點

| 端點 | 說明 |
|------|------|
| `POST /api/auth/register` | 註冊新用戶 |
| `POST /api/auth/login` | 用戶登入 |

### 文章內容增強

| 語法 | 效果 |
|------|------|
| `# 標題` | 大標題 h2 |
| `## 副標題` | h3 |
| `> 引用` | 引用區塊 |
| `- 列表` | 無序列表 |
| `1. 數字` | 有序列表 |

### 示例文章數量

| 版本 | 文章數量 |
|------|----------|
| ver1.0 | 0 (用戶自行創建) |
| ver2.0 | 3 篇 |
| ver3.0 | 11 篇 |
